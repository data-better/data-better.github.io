1. 시계열분석 개요

1.1. 시계열의 정의
□ 시계열(time series) : 시간 t에 따른 확률변수 들의 집합 { }
- 시계열은 확률과정으로 확률공간에서 정의되는 여러 확률변수의 집합
- 실제 관측되는 시계열 자료 : 확률과정의 표본경로
□ 시계열은 다양한 분야에 구해짐 : 국가통계는 대체로 시계열
- 경제 시계열 : 국내총생산(GDP), 소비자물가지수, 수출액, 주가지수, 환율
- 물리 시계열 : 강수량, 기온, 태양의 흑점, 지진파
- 보건, 의학 : 독감 감염자 수 , 혈압측정, 뇌파
- 경영 시계열 : 상품의 판매량, 상품 광고와 판매량 등
- 인구 시계열 : 총인구, 농가수
- 통신 공학 또는 공학과 관련된 시계열 : (0,1)-확률과정, 음성파
